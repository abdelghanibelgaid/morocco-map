# morocco-map
Reference geographic data and compatibility tracking for Morocco maps across open-source ecosystems.

## Data provenance
The GeoJSON file in `maps/morocco.geojson` was manually authored by Abdelghani Belgaid using geojson.io.

No third-party geometry was copied, imported, or derived to create the coordinate geometry. geojson.io was used only as a manual drawing and editing tool.

The resulting polygons were manually reviewed and normalized for Morocco’s current 12-region administrative structure.

Region names and identifiers were normalized for compatibility:

* MA-01 through MA-12
* 12 first-level administrative regions
* WGS84 / EPSG:4326 GeoJSON
